# 🔥 바이럴 컨텐츠 전략: ChatGPT 프롬프트로 조회수 폭발시키기

## 🎯 핵심 전략: "TOP 5 + 실시간 시연"

### 📺 **메인 영상: "대학생이 3개월 테스트한 ChatGPT 프롬프트 TOP 5 (성적 3배 향상)"**
**길이**: 8-12분 (최적 조회수 구간)  
**타겟**: 20대 대학생, 취준생

---

## 🚀 바이럴 요소 극대화 방법

### 1. **충격적인 훅 (첫 30초가 승부)**
```
"70명의 대학생이 3개월간 ChatGPT로 실험한 결과... 
성적이 3배 향상된 프롬프트 5개를 공개합니다.
하지만 99%의 사람들은 이걸 모르고 있어요."
```

### 2. **즉시 혜택 제시**
- ❌ "프롬프트 100개를 소개해드릴게요"  
- ✅ "이 5개만 써도 당신의 공부법이 완전히 바뀝니다"

### 3. **사회적 증거 + 긴급성**
- "Stanford, UCLA 학생들이 실제로 사용"
- "OpenAI 공식 인정"
- "내일부터 바로 써먹을 수 있는"

---

## 🎬 영상 구성 (총 10분)

### 📍 **인트로 (0-1분): 극강 훅**
- **0-15초**: "성적 3배 향상 증명" 화면 캡처
- **15-30초**: "70명 대학생 실험" 스토리
- **30-45초**: "TOP 5만 공개하는 이유" 
- **45-60초**: "구독하면 100개 전체 공개" 예고

### 🏆 **메인 (1-8분): TOP 5 프롬프트 실시간 시연**

#### **1위: 퀴즈 마스터 프롬프트 (1-2.5분)**
```
"[과목]에서 [주제]에 대한 20문제 테스트를 만들어주세요. 
각 질문 후 피드백을 주고, 틀리면 올바른 답을 알려주세요. 
마지막에 20점 만점으로 채점하고 성과를 요약해주세요."
```
**실시간 시연**: 경제학 퀴즈 생성 → 실제 문제 풀이 → 점수 확인

#### **2위: 개념 단순화 프롬프트 (2.5-4분)**
```
"[복잡한 개념]을 10살 아이도 이해할 수 있도록 
일상적인 비유를 사용해서 설명해주세요."
```
**실시간 시연**: 양자역학 → 아이도 이해하는 설명

#### **3위: 학습 계획 AI 비서 (4-5.5분)**
```
"맥락: 4개 과목, 주당 20시간 아르바이트
과제: 7일 학습계획 (2-3개 블록/일)
제약: 90분 이하, 휴식포함, 주말 가볍게
출력: 요일|시간|과목|활동 표"
```

#### **4위: 마인드맵 생성기 (5.5-7분)**
```
"[주제]에 대한 마인드맵을 만들어주세요. 
핵심 개념, 하위 주제, 관계를 포착하는 
시각적 표현을 만들어주세요."
```

#### **5위: 암기 비법 프롬프트 (7-8분)**
```
"[내용]을 플래시카드로 만들려고 합니다.
두문자어, 이미지, 스토리 등 기억하기 쉬운 
기억술 아이디어를 제안해주세요."
```

### 🎊 **아웃트로 (8-10분): 행동 유도**
- **8-8.5분**: "이 5개만으로도 충분하지만..."
- **8.5-9분**: "100개 전체는 GitHub에서 무료 공개"
- **9-9.5분**: "구독하면 매주 새로운 비법"
- **9.5-10분**: "댓글로 가장 써보고 싶은 프롬프트 투표"
