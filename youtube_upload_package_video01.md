- [ ] 썸네일 3개 버전 준비 (A/B/C 테스트용)
- [ ] 제목 키워드 최적화 확인
- [ ] 설명란 링크 작동 테스트
- [ ] 태그 20개 이상 설정
- [ ] 자막 파일 업로드 (.srt)
- [ ] 엔드스크린 설정 (구독 + 다음 영상)
- [ ] 카드 삽입 (GitHub 링크)

### **업로드 후 모니터링**
- [ ] 첫 1시간 댓글 적극 응답
- [ ] 24시간 내 조회수 1K+ 목표
- [ ] 좋아요율 8%+ 유지 확인
- [ ] GitHub 트래픽 증가 모니터링
- [ ] 썸네일 CTR 분석 후 교체 결정

### **성과 분석 포인트**
- [ ] 시청 유지율 (특히 첫 15초)
- [ ] 클릭률 (CTR) 
- [ ] 댓글 참여도
- [ ] 외부 트래픽 (GitHub)
- [ ] 구독자 전환율
