# 📚 ChatGPT Prompts 100 Collection

> 🌟 A comprehensive guide of **100 ChatGPT prompts** voted by 70 college students from **OpenAI ChatGPT Lab**, organized in both Korean and English.

## 🎯 Project Overview

This repository systematically organizes the top 100 ChatGPT prompts for students, collected from **OpenAI ChatGPT Lab** and various educational resources.

### ✨ Features
- 📖 **100 Verified Prompts**: Voted by college students
- 🌐 **Bilingual Support**: Korean + English complete translations
- 📁 **Categorized Organization**: 8 major categories
- 🎥 **YouTube Integration**: Video tutorials included
- 🔧 **Practical Templates**: Ready-to-copy formats

## 🚀 Quick Start

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/chatgpt-prompts-100.git
cd chatgpt-prompts-100
```

### 2. Choose Language
- Korean: Check `korean/` folder
- English: Check `english/` folder

### 3. Explore Categories
- 📚 **Study**: Quizzes, concept explanation, study planning
- 💼 **Career**: Resume, interviews, career exploration
- 🏠 **Life**: Time management, stress management, budgeting
- 📝 **Note-taking**: Lecture notes, summaries
- 🔬 **Research**: Data analysis, literature review
- 💡 **Creative**: Brainstorming, problem solving
- 📊 **Presentation**: Slides, presentation skills
- 🤝 **Collaboration**: Teamwork, project management
