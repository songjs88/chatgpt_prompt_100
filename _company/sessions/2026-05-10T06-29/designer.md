## 🚀 다다직구 CTR 극대화 디자인 마스터 가이드라인 V1.0 (최종)

이 가이드라인은 **'후크 우선 배치와 극단적 대비 원칙'**을 기반으로 하며, 시청자의 스크롤을 멈추고 클릭을 유도하는 데 모든 시각 자원을 집중합니다.

### 1. 디자인 핵심 원칙 (The Core Philosophy)

*   **원칙:** 후크(Hook)가 시각적 요소의 **70% 이상**을 차지해야 한다.
*   **전략:** 극단적인 색상 대비와 명확한 구도 배치를 통해 인지 부하를 최소화하고 감정적 반응을 즉시 유발한다.

### 2. 모바일 최적화 템플릿 (Mobile Optimized Template)

모든 디자인은 **9:16 비율**에 맞춰 제작하며, 텍스트는 화면 중앙 상단(후크 영역)에 집중 배치됩니다.

| 요소 | 위치 및 크기 지침 | 시각 전략 (극단적 대비 적용) |
| :--- | :--- | :--- |
| **배경 (Background)** | 전체 화면 (9:16) | **[핵심]** 배경은 단색(Solid Color) 또는 매우 단순한 질감(Texture)만 사용. 복잡한 이미지는 지양하여 시선이 텍스트에 집중되도록 한다. |
| **후크 텍스트 (Hook Text)** | 화면 상단 1/3 지점 (Top 30%) | **[최고 대비]** 배경과 완전히 반대되는 색상(예: 검은색 배경에 네온 옐로우 또는 시안)을 사용하여 즉각적인 주목도를 확보한다. |
| **핵심 메시지 (Main Point)** | 화면 중앙 (Middle 40%) | **[명확성]** 가장 중요한 정보만 최소한의 폰트(굵고 단순한 산세리프체, 예: Pretendard Bold)로 배치. 3~5단어 이내로 압축한다. |
| **감정 유발 요소 (Emotion Trigger)** | 화면 하단 2/3 지점 (Bottom 60%) | **[대비 강조]** 배경과 대비되는 색상을 사용하여 시각적 무게중심을 확보하고, 행동(Call to Action)의 긴급성을 부여한다. |
| **CTA 영역 (Call to Action)** | 화면 최하단 바 (Bottom Bar) | **[행동 유도]** 눈에 띄는 버튼 형태나 화살표 아이콘을 사용하여 시청자가 다음 행동(클릭/댓글)을 하도록 명확하게 지시한다. |

### 3. 컬러 및 타이포 시스템 (Color & Type System)

이 시스템은 모든 영상에서 일관성을 유지하여 브랜드 인지도를 높입니다.

| 요소 | 색상 코드 (Hex) | 적용 목적 | 폰트 스타일 (예시) |
| :--- | :--- | :--- | :--- |
| **Primary Hook Color** | `#FF0055` (강렬한 마젠타) | 후크 텍스트 및 CTA 강조. 클릭 유도. | Impact, 또는 고딕 계열의 매우 두꺼운 폰트 |
| **Contrast Background** | `#111111` (딥 블랙) | 배경색으로 사용 시 극단적인 대비 효과 창출. | N/A |
| **Secondary Text** | `#FFFFFF` (순백) | 보조 설명 및 본문 정보. | Pretendard Medium |

### 4. 최종 실행 전략 (Execution Strategy)

1.  **후크 스크립트 적용:** Writer가 제공한 **최우선 후크 스크립트**를 화면 상단에 배치하고, 해당 스크립트의 핵심 감정(놀람, 분노, 호기심 등)을 Primary Hook Color로 강조합니다.
2.  **시각 자원 집중:** 모든 복잡한 상품 이미지는 최소화하고, **'문제 제기 → 충격적 결과 → 즉각적인 해결책 제시'**의 흐름에 맞춰 텍스트와 대비 색상으로만 구성합니다.
3.  **템플릿 적용:** 위 표를 기반으로 Figma 또는 편집 툴에서 직접 레이아웃을 구현하여, **모바일 환경에서 1초 안에 시청자의 시선을 잡아채도록** 디자인합니다.
