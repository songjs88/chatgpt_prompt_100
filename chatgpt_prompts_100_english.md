## Study Prompts

### 1. Quiz Creation
```
Create a 20-question test on [topic] for [subject]. After each of my responses, provide feedback and tell me the correct answer if I'm wrong. At the end, grade me out of 20 and summarize my performance.
```

### 2. Complex Concept Simplification
```
Explain [concept/topic] in a way that a 10-year-old could understand. Use everyday analogies in your explanation.
```

### 3. Study Plan Development
```
Context: I have 4 classes, each with weekly readings and quizzes. I also have a part-time job (20 hrs/week).
Task: Build a 7-day study plan with 2-3 focused blocks per day.
Constraints: Sessions ≤90 minutes, include breaks, weekends lighter, evenings free after 9 PM
Output: Table with Day | Time | Subject | Activity
```

### 4. Flashcard Generation
```
Create flashcards for [subject] [topic]. Put questions on the front and answers on the back. Use simple phrases that are easy to memorize.
```

### 5. Research Question Development
```
Formulate a research question on the societal implications of [topic], ensuring clarity and alignment with your study's objectives.
```

### 6. Literature Review Conduct
```
Summarize key research on [topic], synthesizing existing knowledge and identifying research gaps.
```

### 7. Concept Visualization
```
Create a mind map for [concept]. Make a visual representation that captures key concepts, subtopics, and their relationships.
```

### 8. Discussion Prompt Generation
```
I need thought-provoking discussion prompts for my lesson on [topic].
```

### 9. Learning Style Customization
```

```

### 10. Concept Example Creation
```

```

### 11. Analogy Generation
```
I'm writing a note called '[concept name]'. I'm trying to come up with an analogy that can explain this concept in words a ten-year-old could understand. What are some ideas?
```

### 12. Summary Writing

[text]
```

### 13. Memory Technique Suggestions
```

## Usage Tips

### General Advice for ChatGPT Usage:

1. **Be Specific**: Present specific situations and requirements rather than vague questions.

2. **Provide Context**: Provide related information such as grade level, major, current situation for more appropriate responses.

3. **Step-by-Step Approach**: Request complex tasks in multiple steps.

4. **Use Feedback**: Ask additional questions or request modifications to ChatGPT's responses.

5. **Critical Thinking**: Don't blindly trust ChatGPT's responses; verify and supplement them.

### Precautions:

- ChatGPT was trained on data up to 2021, so separate verification is needed for the latest information.
- Maintain academic integrity and be careful not to engage in cheating on assignments or exams.
- ChatGPT is just a tool and cannot replace critical thinking and personal learning efforts.
