## 📊 메인 콘텐츠 Part 1: 프로젝트 배경 (2:30 - 6:00)

### [화면: OpenAI ChatGPT Lab 설명 인포그래픽]
**음성**: 자, 그럼 이 프로젝트가 어떻게 시작된 건지부터 설명드릴게요.

2025년 초, OpenAI에서는 "ChatGPT Lab"이라는 특별한 프로그램을 시작했습니다. 전 세계 50개 이상의 대학교에서 70명의 학생들을 선발했어요.

### [화면: 참여 대학교 리스트]
**음성**: 스탠포드, 펜실베니아 대학교, 오하이오 주립대, UCLA, BYU... 정말 쟁쟁한 대학들이죠.

### [화면: 실험 과정 애니메이션]
**음성**: 이 학생들에게 주어진 미션은 무엇이었을까요? 바로 "가장 유용한 ChatGPT 프롬프트를 찾아내라"는 것이었습니다.

3개월 동안 이들은:
- 매주 새로운 프롬프트를 테스트하고
- 서로의 프롬프트를 공유하고
- 실제 효과를 검증하고
- 최종적으로 투표를 통해 순위를 매겼습니다

### [화면: 최종 결과 발표]
**음성**: 그 결과가 바로 이 100개 프롬프트입니다. 이건 단순히 누군가 만든 프롬프트가 아니라, **실제로 검증된, 효과가 입증된** 프롬프트라는 뜻이에요.

## 🗂️ 메인 콘텐츠 Part 2: 8개 카테고리 분석 (6:00 - 11:00)

### [화면: 카테고리 분류 애니메이션]
**음성**: 이 100개 프롬프트는 8개의 주요 카테고리로 나뉩니다. 하나씩 살펴볼게요.

### [화면: 📚 학습 아이콘 + 15개]
**음성**: **첫 번째, 학습 프롬프트 15개**  
퀴즈 생성, 개념 설명, 학습 계획... 공부하는 학생들이 가장 많이 사용한 프롬프트들이에요. 다음 주에 이것만 깊게 다룰 예정입니다.

### [화면: 💼 진로 아이콘 + 15개]
**음성**: **두 번째, 진로 프롬프트 15개**  
이력서 검토, 면접 준비, 진로 탐색... 요즘 취업이 어려운 시대에 정말 실용적인 프롬프트들이죠.

### [화면: 🏠 생활 아이콘 + 15개]
**음성**: **세 번째, 생활 프롬프트 15개**  
시간 관리, 스트레스 관리, 예산 관리... 대학생활의 모든 고민을 해결해주는 프롬프트들입니다.

### [화면: 나머지 카테고리들 빠르게 스캔]
**음성**: 그 외에도 노트 정리, 연구 분석, 창의적 사고, 발표 준비, 팀워크까지... 정말 학생이 필요한 모든 상황을 다 커버하고 있어요.

### [화면: 통계 차트]
**음성**: 재미있는 건, 학생들이 가장 많이 사용한 카테고리는 당연히 '학습' 프롬프트였지만, 의외로 '생활 관리' 프롬프트의 만족도가 가장 높았다는 점입니다.

## 🎯 메인 콘텐츠 Part 3: 실제 프롬프트 시연 (11:00 - 15:30)

**음성**: 자, 이제 말로만 하지 말고 실제로 보여드릴게요. 3개의 프롬프트를 라이브로 시연해보겠습니다.

### [화면: ChatGPT 화면 녹화 시작]

#### 시연 1: 퀴즈 생성 프롬프트 (11:00 - 12:30)
**음성**: 첫 번째는 1위를 차지한 퀴즈 생성 프롬프트입니다.

*[화면: 프롬프트 입력]*
```
경제학에서 수요와 공급에 대한 20문제 테스트를 만들어주세요. 각 질문에 대한 내 답변 후에는 피드백을 주고, 틀린 부분이 있다면 올바른 답을 알려주세요. 마지막에 20점 만점으로 채점하고 성과를 요약해 주세요.
```

**음성**: 보세요! 정말 완벽한 퀴즈가 생성되었습니다. 그냥 "퀴즈 만들어줘"라고 하는 것과는 차원이 다르죠? 

핵심은 **구체적인 지시사항**입니다. 몇 문제인지, 어떤 피드백을 원하는지, 최종 결과를 어떻게 정리해달라는지... 모든 걸 명확히 알려주는 거예요.

#### 시연 2: 복잡한 개념 단순화 (12:30 - 13:45)
**음성**: 두 번째는 복잡한 개념을 쉽게 설명해주는 프롬프트입니다.

*[화면: 프롬프트 입력]*
```
양자역학을 10살 아이도 이해할 수 있도록 간단하게 설명해주세요. 일상적인 비유를 사용해서 설명해주세요.
```

**음성**: 와! 정말 10살 아이도 이해할 수 있을 것 같네요. "고양이가 상자 안에 있으면서 동시에 없을 수도 있다"는 설명... 슈뢰딩거의 고양이를 이렇게 쉽게 설명할 수 있다니!

#### 시연 3: 마인드맵 생성 (13:45 - 15:30)
**음성**: 마지막은 제가 개인적으로 가장 놀란 프롬프트입니다.

*[화면: 프롬프트 입력]*
```
머신러닝에 대한 마인드맵을 만들어주세요. 핵심 개념, 하위 주제, 그리고 그들 간의 관계를 포착하는 시각적 표현을 만들어주세요.
```

**음성**: 정말 놀랍지 않나요? 텍스트로 마인드맵을 만들어주고 있어요. 이걸 MindMeister나 XMind 같은 도구로 옮기면 완벽한 마인드맵이 됩니다.

### [화면: 시연 결과 정리]
**음성**: 보셨죠? 같은 질문이라도 프롬프트를 어떻게 작성하느냐에 따라 결과가 완전히 달라집니다. 이게 바로 "프롬프트 엔지니어링"의 힘입니다.
