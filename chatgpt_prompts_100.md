## 학습 프롬프트

### 1. 퀴즈 만들기
```
[과목]에서 [주제]에 대한 20문제 테스트를 만들어주세요. 각 질문에 대한 내 답변 후에는 피드백을 주고, 틀린 부분이 있다면 올바른 답을 알려주세요. 마지막에 20점 만점으로 채점하고 성과를 요약해 주세요.
```

### 2. 복잡한 개념 단순화
```
[개념/주제]를 10살 아이도 이해할 수 있도록 간단하게 설명해주세요. 일상적인 비유를 사용해서 설명해주세요.
```

### 3. 학습 계획 수립
```
맥락: 저는 4개 과목을 듣고 있고, 매주 읽을 자료와 퀴즈가 있습니다. 또한 주당 20시간 아르바이트를 하고 있습니다.
과제: 하루에 2-3개의 집중 블록으로 구성된 7일 학습 계획을 세워주세요.
제약사항: 세션은 90분 이하, 휴식시간 포함, 주말은 가볍게, 오후 9시 이후는 자유시간
출력: 요일 | 시간 | 과목 | 활동 형태의 표
```

### 4. 플래시카드 생성
```
[과목] [주제]에 대한 플래시카드를 만들어주세요. 앞면에는 질문을, 뒷면에는 답을 적어주세요. 기억하기 쉽도록 간단한 문구를 사용해주세요.
```

### 5. 연구 질문 개발
```
[주제]의 사회적 영향에 대한 연구 질문을 만들어주세요. 명확성과 연구 목표와의 일치성을 보장해주세요.
```

### 6. 문헌 검토 수행
```
[주제]에 대한 핵심 연구를 요약하고, 기존 지식을 종합하여 연구 공백을 찾아주세요.
```

### 7. 개념 시각화
```
[개념]에 대한 마인드맵을 만들어주세요. 핵심 개념, 하위 주제, 그리고 그들 간의 관계를 포착하는 시각적 표현을 만들어주세요.
```

### 8. 토론 프롬프트 생성
```
[주제]에 대한 수업을 위한 사고를 자극하는 토론 프롬프트가 필요합니다.
```

### 9. 학습 스타일 맞춤
```
저는 [학습 스타일/요구사항]을 가진 학생입니다. 이 학습 스타일에 맞는 구체적인 자료나 도구를 추천해주세요.
```

### 10. 개념 예시 만들기
```
'[개념명]'이라는 노트를 작성하고 있습니다. 이 개념을 예시로 보여줄 수 있는 예제를 생각해보고 있습니다. 아래 노트 내용을 바탕으로 어떤 아이디어가 있을까요?
```

### 11. 비유 생성
```
'[개념명]'이라는 노트를 작성하고 있습니다. 10살 아이가 이해할 수 있는 말로 이 개념을 설명할 수 있는 비유를 생각해보고 있습니다. 어떤 아이디어가 있을까요?
```

### 12. 요약 작성
```
[요약이 필요한 이유]를 위해 다음 텍스트의 요약을 [원하는 형식]으로 만들어주세요. [특정 아이디어]에 집중해주세요.

다음은 텍스트입니다:
[텍스트]
```

### 13. 암기 기법 제안
```
이 노트 내용으로 플래시카드를 만들려고 합니다: [노트 내용]
두문자어, 이미지, 캐릭터화, 스토리 등 기억하기 쉽게 만들 수 있는 기억술 아이디어가 있을까요?
```

### 14. 스터디 가이드 생성
```
[과목]의 [주제] 시험을 준비하고 있습니다. 주요 개념, 핵심 용어, 연습 문제를 포함한 종합적인 스터디 가이드를 만들어주세요.
```

### 15. 과제 분석
```
[과제 문제]를 작업하고 있습니다. 지금까지 [작업한 내용]을 완성했습니다. 어디서부터 더 진행해야 할지 아이디어를 도와줄 수 있나요?
```

## 진로 프롬프트

### 16. 이력서 검토
```
[직무/분야]에 지원하기 위한 이력서를 검토해주세요. 개선할 점과 추가해야 할 기술이나 경험에 대한 제안을 해주세요.
```

### 17. 면접 준비
```
[직무]의 [회사] 면접을 준비하고 있습니다. 일반적인 면접 질문과 모범 답변을 준비하는 데 도움을 주세요.
```

### 18. 커버레터 작성
```
[회사]의 [직무]에 지원하기 위한 커버레터를 작성해주세요. 제 배경: [간단한 배경 설명]
```

### 19. 진로 탐색
```
[관심 분야/전공]에 관심이 있는 학생입니다. 이 분야에서 가능한 진로 경로와 필요한 기술들을 알려주세요.
```

### 20. 네트워킹 조언
```
[산업/분야]에서 전문가들과 네트워킹하는 방법에 대한 조언을 해주세요. 온라인과 오프라인 모두 포함해서 알려주세요.
```

### 21. 대학원 준비
```
[전공]으로 대학원에 진학하려고 합니다. 지원서 작성, 추천서 요청, 입학 시험 준비에 대한 가이드를 제공해주세요.
```

### 22. 인턴십 찾기
```
[분야]에서 인턴십을 찾고 있습니다. 어떤 회사들을 고려해야 하고, 어떻게 지원해야 하는지 조언해주세요.
```

### 23. 기술 개발 계획
```
[직무/산업]에서 성공하기 위해 개발해야 할 기술들을 나열하고, 각 기술을 배울 수 있는 방법을 제안해주세요.
```

### 24. 포트폴리오 구성
```
[분야] 포트폴리오를 만들고 있습니다. 어떤 프로젝트들을 포함해야 하고, 어떻게 구성해야 하는지 조언해주세요.
```

### 25. 진로 변경 조언
```
현재 [현재 전공/분야]를 공부하고 있지만 [관심 분야]로 진로를 바꾸고 싶습니다. 전환을 위한 단계적 계획을 세워주세요.
```

## 생활 프롬프트

### 26. 시간 관리
```
대학생으로서 학업, 아르바이트, 개인 시간의 균형을 맞추는 효과적인 시간 관리 전략을 제안해주세요.
```

### 27. 스트레스 관리
```
시험 기간 동안 스트레스를 관리하는 건강한 방법들을 알려주세요. 실용적이고 즉시 실행 가능한 방법들로 부탁합니다.
```

### 28. 예산 관리
```
한 달 생활비가 [금액]원인 대학생을 위한 예산 계획을 세워주세요. 식비, 교통비, 학용품비 등을 포함해서 짜주세요.
```

### 29. 건강한 식단
```
맥락: 기숙사에 살고 있고, 가스레인지는 없고 전자레인지와 작은 냉장고만 있습니다.
과제: 일주일 5일 식단을 5만원 이하로, 고단백질로 계획해주세요.
제약사항: 하루 2000칼로리 이하, 15분 이하 조리시간
출력: 요일 | 식사 | 재료 | 조리법 표
```

### 30. 운동 루틴
```
기숙사 방에서 할 수 있는 30분 운동 루틴을 짜주세요. 헬스장이나 특별한 기구 없이 할 수 있는 운동으로 부탁합니다.
```

### 31. 수면 개선
```
늦은 시간까지 공부하는 대학생을 위한 수면의 질을 개선하는 방법을 알려주세요.
```

### 32. 사회적 관계
```
대학에서 새로운 친구를 사귀고 의미 있는 관계를 형성하는 방법에 대한 조언을 해주세요.
```

### 33. 여행 계획
```
[예산]원으로 [기간] 동안 [목적지]를 여행하는 계획을 세워주세요. 교통, 숙박, 식사, 관광을 포함해서 짜주세요.
```

### 34. 취미 개발
```
학업 스트레스 해소를 위한 새로운 취미를 추천해주세요. 시간과 비용이 많이 들지 않는 것들로 부탁합니다.
```

### 35. 갈등 해결
```
룸메이트/친구와의 갈등을 건설적으로 해결하는 방법을 알려주세요. 의사소통 기술도 포함해서 설명해주세요.
```

## 노트 정리 프롬프트

### 36. 강의 노트 정리
```
다음 강의 노트를 구조화된 형식으로 정리해주세요. 주요 포인트, 하위 개념, 예시를 명확히 구분해서 정리해주세요:
[강의 노트 내용]
```

### 37. 요점 추출
```
이 긴 텍스트에서 가장 중요한 5가지 핵심 포인트를 추출해주세요:
[텍스트 내용]
```

### 38. 구조화된 요약
```
다음 자료를 학습에 최적화된 구조화된 형식으로 간결한 노트를 작성해주세요. 각 노트 사이에는 공간을 두어 주세요:
[자료 내용]
```

### 39. 비교 분석
```
[개념 A]와 [개념 B]의 유사점과 차이점을 비교 분석하여 표로 정리해주세요.
```

### 40. 용어 정리
```
[주제]와 관련된 핵심 용어들을 정의와 함께 정리해주세요. 각 용어마다 간단한 예시도 포함해주세요.
```

## 연구 및 분석 프롬프트

### 41. 데이터 해석
```
WHO 최신 데이터를 바탕으로 여러 국가의 식습관과 기대수명 간의 상관관계를 해석해주세요.
```

### 42. 비판적 분석
```
도시 지역에서의 태양 에너지 도입 가능성을 비용, 효율성, 공간 제약을 고려하여 비판적으로 평가해주세요.
```

### 43. 정보 종합
```
원격 근무가 생산성과 직원 복지에 미치는 영향에 대한 최근 연구 결과들을 종합해주세요.
```

### 44. 연구 방법론
```
[주제]에 대해 학생들이 연구할 때 사용할 수 있는 주요 접근법, 연구 방법, 데이터 수집 기법을 제안해주세요.
```

### 45. 가설 설정
```
[연구 주제]에 대한 연구 가설을 설정하고, 이를 검증하기 위한 연구 설계를 제안해주세요.
```

### 46. 문헌 검토
```
[주제]에 대한 문헌 검토를 위해 신뢰할 만한 출처와 정보를 제공해주세요.
```

### 47. 연구 윤리
```
[연구 분야]에서 연구를 수행할 때 고려해야 할 윤리적 사항들을 설명해주세요.
```

### 48. 데이터 시각화
```
[데이터 유형]을 효과적으로 시각화하기 위한 차트나 그래프 유형을 추천하고, 각각의 장단점을 설명해주세요.
```

### 49. 연구 질문 개선
```
다음 연구 질문을 더 구체적이고 연구 가능하도록 개선해주세요: [연구 질문]
```

### 50. 연구 계획서
```
[주제]에 대한 소규모 연구 프로젝트의 계획서를 작성해주세요. 목적, 방법, 일정, 예상 결과를 포함해주세요.
```

## 창의적 사고 프롬프트

### 51. 브레인스토밍
```
[프로젝트/문제]를 작업하고 있습니다. 이 프로젝트/문제를 해결하기 위한 혁신적인 아이디어와 잠재적 접근법을 생성하는 데 도움을 줄 수 있나요?
```

### 52. 창의적 글쓰기
```
[장르]로 [주제]에 대한 창의적인 이야기를 시작해주세요. 흥미로운 캐릭터와 갈등 상황을 포함해주세요.
```

### 53. 문제 해결
```
다음 문제가 있습니다: [문제]. 이 문제를 효과적으로 분석하고 해결하기 위해 사용할 수 있는 다양한 접근법, 방법론, 프레임워크를 브레인스토밍하는 데 도움을 줄 수 있나요?
```

### 54. 대안적 관점
```
[주제/문제]를 역방향 관점에서 생각해봅시다.
```

### 55. 혁신적 아이디어
```
[전통적인 방법/시스템]을 개선하거나 완전히 새롭게 재설계한다면 어떤 아이디어가 있을까요?
```

### 56. 시나리오 플래닝
```
[주제/상황]에 대한 가상의 시나리오를 생성해주세요. 세부사항과 잠재적 영향을 포함해주세요.
```

### 57. 아이디어 연결
```
[개념 A]와 [개념 B] 사이의 예상치 못한 연결고리나 공통점을 찾아주세요.
```

### 58. 창의적 응용
```
[기존 기술/방법]을 [새로운 분야]에 어떻게 창의적으로 응용할 수 있을까요?
```

### 59. 미래 예측
```
현재 [트렌드/기술]이 10년 후에는 어떻게 발전되어 있을지 창의적으로 예측해주세요.
```

### 60. 창의적 조합
```
[요소 1], [요소 2], [요소 3]을 조합하여 새로운 아이디어나 솔루션을 만들어주세요.
```

## 시험 준비 프롬프트

### 61. 시험 전략
```
[과목] 시험을 [기간] 후에 봅니다. 효과적인 시험 준비 전략과 일정을 세워주세요.
```

## 사용 팁

### ChatGPT 활용을 위한 일반적인 조언:

1. **구체적으로 요청하기**: 모호한 질문보다는 구체적인 상황과 요구사항을 제시하세요.

2. **맥락 제공하기**: 학년, 전공, 현재 상황 등 관련 정보를 함께 제공하면 더 적절한 답변을 받을 수 있습니다.

3. **단계별 접근**: 복잡한 작업은 여러 단계로 나누어 요청하세요.

4. **피드백 활용**: ChatGPT의 답변에 대해 추가 질문이나 수정 요청을 하세요.

5. **비판적 사고**: ChatGPT의 답변을 무조건 신뢰하지 말고 검증하고 보완하세요.

### 주의사항:

- ChatGPT는 2021년까지의 데이터로 훈련되었으므로 최신 정보는 별도로 확인이 필요합니다.
- 학술적 무결성을 유지하고, 과제나 시험에서 부정행위가 되지 않도록 주의하세요.
- ChatGPT는 도구일 뿐이며, 비판적 사고와 개인의 학습 노력을 대체할 수 없습니다.
